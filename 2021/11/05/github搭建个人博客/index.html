创建github仓库

本地搭建博客网站

  1. 安装nodejs

  2. 安装hexo

  3. 创建博客

  4. 测试

    本地博客网站创建成功

配置SSH Key

  1. 在Powershell（cmd）输入 ssh-keygen -t rsa -C “your email”，指定文件名等信息

  2. 此时会生成私钥文件和公钥文件，后缀为pud的为公钥文件复制id_rsa.pub文件中的内容

  3. 在GitHub设置中新建ssh key并填入生成公钥内容

  4. 配置HOST文件，输入以下内容

    Host 为了代替下面的主机的名字
    HostName是真实的域名github.com，可以通过Host代替它
    IdentityFile是用来验证身份的，这里放置一个私钥相当于登录密码

  5. 测试是否成功

上传到github

  1. 安装插件

  2. 修改配置文件_config.yml中deploy部分

  3. 将搭建的网站push到GitHub上
